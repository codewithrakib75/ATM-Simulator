# ATM Simulator (Basic)

A simple ATM simulator program written in C. This project allows users to check balance, deposit money, withdraw money, and exit using a menu-driven interface.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven system

## Concepts Used
- Variables
- Loop (do-while)
- switch-case
- Conditional statements
- Input/output functions
